# 🧠 Patra’s E-Commerce Sales Dashboard – Power BI

## 📊 Project Overview
This Power BI project analyzes **e-commerce sales performance** across multiple regions, product categories, and payment modes.  
The dashboard provides actionable insights into **total sales, profit, quantity sold, and average order value (AOV)**, helping businesses track KPIs and improve decision-making.

---

## 🧩 Objectives
- Visualize sales and profit trends across different **states and categories**.  
- Identify **top-performing products, customers, and payment methods**.  
- Provide an interactive way to explore **quarterly and regional performance**.  
- Enable management to **make data-driven decisions** through a clean and dynamic interface.

---

## ⚙️ Tools & Technologies Used
- **Power BI Desktop**
- **DAX (Data Analysis Expressions)**
- **Power Query (Data Cleaning & Transformation)**
- **Excel** (for data source)
- **Data Modeling and Relationships**

---

## 📈 Key Features
- **Interactive Dashboard:** Filters, slicers, and drill-through options for deeper insights.  
- **Dynamic KPIs:** Total Sales, Profit, Quantity, and AOV.  
- **Custom Visuals:** Bar chart, donut chart, pie chart, line chart, scatter chart, and map.  
- **Regional Insights:** Highlights top 4 states contributing to revenue.  
- **Customer Analysis:** Identifies most valuable customers and preferred payment modes.

---

## 💡 Key Insights
- Maharashtra and Madhya Pradesh contribute the highest sales volume.  
- Clothing category drives **63%** of total sales.  
- Cash on Delivery (COD) accounts for **44%** of total transactions.  
- Highest profit recorded in **Printers and Bookcases** sub-categories.

---

## 🧾 Dataset
- **Source:** Simulated e-commerce sales dataset  
- **Records:** ~1,200 transactions  
- **Fields:** Order ID, Customer, State, Category, Sub-Category, Sales, Profit, Quantity, Payment Mode, Date  

---

## 🚀 How to Use
1. Download the `.pbix` file from this repository.  
2. Open it using **Power BI Desktop (latest version)**.  
3. Explore interactive visuals using filters, slicers, and drill-through options.
